## Projects

**FakeMail.ai**
AI-generated image detector. Combines CLIP embeddings with a frequency-domain CNN to catch artifacts that pure vision transformers miss.
- 94% validation accuracy
- Hybrid semantic + frequency-domain feature pipeline

**Salary Tax Optimizer (FY 25-26)**
Rules engine for Indian income tax paired with an LP optimizer for allocating deductions/investments, plus RAG over the Income-tax Act for natural-language queries.
- Optimization via scipy / PuLP
- RAG layer grounded directly in statute text, not summaries

**Drivera / DriveScore AI**
Driving safety scorer that turns raw trip data into a readable risk score.
- React frontend, FastAPI backend
- Event-based scoring (harsh braking, speeding, phone use, etc.)

**PQ-REAPER**
Post-quantum cryptography vulnerability scanning engine, built during a DRDO SAG internship.
- Flags classically-secure but quantum-vulnerable crypto usage in codebases
- Built for a defense research context, so accuracy over speed

**EchoSense**
iOS accessibility app for the visually impaired, using Vision + CoreML for real-time scene understanding and spatial audio for directional feedback.
- On-device inference, no cloud round-trip
- Spatial audio cues mapped to object position/distance
